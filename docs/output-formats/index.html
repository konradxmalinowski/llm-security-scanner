Output Formats

Terminal output is always shown. Optionally save structured reports for archiving, diffing, or sharing with --format.

Risk score bands:
0 – 3.9 Low
4 – 6.9 Medium
7 – 10 High
$_
Terminal
always on
Rich table with colour-coded severity rows. Risk score printed in green / yellow / red based on band. Always shown, even when other formats are requested.
.md
Markdown
--format md
Clean table with attack ID, category, name, severity, result, and recommendation. Ideal for pasting into GitHub issues or Notion.
{}
JSON
--format json
Full ScanReport structure including judge_reasoning per finding. Best for programmatic processing, diffing results over time, or feeding into CI pipelines.
.html
HTML
--format html
Self-contained file. Rows colour-coded by severity. Jinja2 autoescape=True ensures attack payloads containing <script> render as escaped text — XSS-safe by design.
